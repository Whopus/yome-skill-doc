# @yome/doc

Microsoft Word skill for [Yome](https://yome.work) agents — covers the
Word for Mac AppleScript surface (file ops + paragraphs + tables, headers /
footers / sections, find / replace, bookmarks, hyperlinks, comments, images /
shapes, styles, lists, table-of-contents, page setup, view, track changes,
print / export). Official.

## Install

```sh
yome skill install github:Whopus/yome-skill-doc
```
